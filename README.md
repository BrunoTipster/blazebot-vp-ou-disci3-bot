# BlazeBotVP

Um bot Telegram avançado para análise de padrões e previsões automáticas com integração ao cassino Blaze.

## Características

- 🤖 Bot Telegram automatizado
- 📊 Análise de padrões de resultados
- 💰 Gerenciamento de banca e apostas
- 🎯 Sistema de sinais com múltiplos níveis de confiança
- 📈 Histórico de resultados e estatísticas
- ⚙️ Integração com Vermelho Engine

## Requisitos

- Python 3.8+
- pip

## Instalação

1. Clone o repositório:
```bash
git clone https://github.com/BrunoTipster/[nome-do-repositorio]
cd [nome-do-repositorio]
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

3. Configure o arquivo `config.ini` com suas credenciais:
```ini
[bot_config]
api_key = seu_token_telegram
chat_id = seu_chat_id

[telegram]
api_id = seu_api_id
api_hash = seu_api_hash
phone = seu_telefone
```

## Uso

Execute o bot:
```bash
python bot22.py
```

## Estrutura de Arquivos

- `bot22.py` - Script principal do bot
- `config.ini` - Configurações e credenciais
- `*_stats.json` - Arquivos de estatísticas
- `*_padroes*.json` - Padrões de análise
- `branco*.json` - Dados de histórico
